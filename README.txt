◇ ゲームの起動方法 ◇
・Windowsをお使いの方
「Windows」フォルダ内にある、「BOTCH-MAN.exe」を実行してください。

・MacOSをお使いの方
「MacOS」フォルダ内にある、「BOTCH-MAN.app」を実行してください。


◇ ルール ◇
キーボードの↑↓←→ボタンを押してキャラクターを動かして、・を全部食べるとステージクリア！
ただし、敵に当たると残機が減ってしまいます。
大きな●を食べるとパワーアップ！
パワーアップ中は敵を倒すことができます。

敵をたくさん倒したり、途中で出現するスペシャルアイテムを食べると、スコアがアップします。
目指せランキング1位！
